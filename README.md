# Compensaciones-por-Tiempo-en-Losa
Esta aplicación permite cargar un archivo CSV con información de reservas y generar un reporte filtrado con:  - Selección automática de columnas relevantes   - Filtro por fecha (rango de fechas)   - Selección de estado de pago (Pagado / No Pagado)   - Cálculo automático del monto a reembolsar según reglas de negocio
